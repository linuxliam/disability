# Troubleshooting Guide

Solutions to common issues with the Disability Advocacy app.

## App Won't Launch

### Issue: App crashes on launch

**Solutions:**
1. **Restart your device** - This resolves many temporary issues
2. **Check system requirements** - Ensure your device meets minimum requirements:
   - iOS 15.0 or later
   - macOS 15.0 or later
3. **Reinstall the app** - Delete and reinstall from the App Store
4. **Check for updates** - Make sure you have the latest version
5. **Free up storage** - Ensure you have enough storage space

### Issue: App opens but shows blank screen

**Solutions:**
1. **Check internet connection** - The app needs internet to load data
2. **Wait a moment** - Initial data loading may take a few seconds
3. **Pull to refresh** - Swipe down to refresh the content
4. **Restart the app** - Close and reopen the app

## Data Loading Issues

### Issue: Resources or events won't load

**Solutions:**
1. **Check internet connection** - Ensure you're connected to Wi-Fi or cellular data
2. **Pull to refresh** - Swipe down on the list to refresh
3. **Check filters** - Make sure no filters are hiding content
4. **Clear search** - If searching, clear the search and try again
5. **Restart the app** - Close and reopen the app

### Issue: Search returns no results

**Solutions:**
1. **Check spelling** - Verify your search terms are spelled correctly
2. **Try different keywords** - Use alternative search terms
3. **Clear filters** - Remove any active filters
4. **Check internet connection** - Ensure you're online
5. **Try a broader search** - Use more general terms

## Calendar Integration Issues

### Issue: Can't add events to calendar

**Solutions:**
1. **Check permissions** - Go to Settings > Privacy & Security > Calendars and ensure the app has permission
2. **Grant permission** - When prompted, allow calendar access
3. **Check calendar app** - Ensure your default calendar app is working
4. **Try again** - After granting permission, try adding the event again

### Issue: Events added but not showing in calendar

**Solutions:**
1. **Check calendar app** - Open your calendar app and verify the event
2. **Check calendar account** - Ensure you're looking at the correct calendar
3. **Sync calendars** - If using iCloud, ensure sync is enabled
4. **Wait a moment** - Calendar sync may take a few seconds

## Favorites Issues

### Issue: Favorites aren't saving

**Solutions:**
1. **Check storage** - Ensure you have enough device storage
2. **Restart the app** - Close and reopen the app
3. **Try again** - Tap the heart icon again to add to favorites
4. **Check profile** - View your profile to see if favorites are there

### Issue: Favorites disappeared

**Solutions:**
1. **Check if you're logged in** - Favorites are stored locally
2. **Restart the app** - Sometimes a restart restores data
3. **Re-add favorites** - If lost, you can re-add them

## Performance Issues

### Issue: App is slow or laggy

**Solutions:**
1. **Close other apps** - Free up device memory
2. **Restart device** - This clears memory and caches
3. **Check storage** - Free up device storage if low
4. **Update the app** - Install the latest version
5. **Update iOS/macOS** - Ensure your OS is up to date

### Issue: App uses too much battery

**Solutions:**
1. **Reduce background activity** - Close the app when not in use
2. **Check for updates** - Newer versions may be more efficient
3. **Reduce refresh frequency** - Don't constantly refresh content
4. **Check device settings** - Review battery usage in Settings

## Display Issues

### Issue: Text is too small or too large

**Solutions:**
1. **Adjust Dynamic Type** - Go to Settings > Display & Brightness > Text Size
2. **App settings** - Check Accessibility Settings in the app
3. **System settings** - Adjust system-wide text size settings

### Issue: Colors are hard to see

**Solutions:**
1. **Enable High Contrast** - Go to Accessibility Settings > High Contrast Mode
2. **Check Dark Mode** - Toggle Dark Mode in system settings
3. **Adjust display settings** - Use system display settings

## macOS-Specific Issues

### Issue: File import/export not working

**Solutions:**
1. **Check file format** - Ensure files are valid JSON format
2. **Check permissions** - Grant file access permissions when prompted
3. **Try different file** - Test with a known-good file
4. **Check file location** - Ensure you have access to the file location

### Issue: Share sheet not appearing

**Solutions:**
1. **Check macOS version** - Ensure you're running macOS 15.0 or later
2. **Try different item** - Share a different resource or event
3. **Restart the app** - Close and reopen the app
4. **Check system settings** - Verify share sheet is enabled in system settings

## iOS-Specific Issues

### Issue: Haptic feedback not working

**Solutions:**
1. **Check system settings** - Go to Settings > Sounds & Haptics
2. **Check app permissions** - Ensure haptics are enabled
3. **Check device** - Some older devices may not support all haptic types
4. **Restart device** - Sometimes a restart fixes haptic issues

### Issue: Pull-to-refresh not working

**Solutions:**
1. **Swipe from top** - Start your swipe from the very top of the list
2. **Pull further** - Pull down more to trigger refresh
3. **Wait for loading** - Give the refresh time to complete
4. **Try again** - Sometimes a second attempt works

## Still Having Issues?

If you're still experiencing problems:

1. **Check for app updates** - Install the latest version
2. **Check system updates** - Update your iOS or macOS
3. **Contact support** - Use the feedback system in the app
4. **Report the issue** - Include:
   - Device model and OS version
   - App version
   - Steps to reproduce
   - Screenshots if possible

## Prevention Tips

To avoid issues:

- **Keep the app updated** - Install updates when available
- **Keep your device updated** - Update iOS/macOS regularly
- **Maintain storage space** - Keep at least 1GB free
- **Restart regularly** - Restart your device weekly
- **Check internet connection** - Ensure stable internet for data loading

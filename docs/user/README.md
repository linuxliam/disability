## About the App

The Disability Advocacy app is designed to support disability advocacy by providing:

- **Resources Library**: Comprehensive database of disability-related resources
- **Events Calendar**: Upcoming workshops, conferences, and events
- **Community Forum**: Connect with others in the disability community
- **Advocacy Tools**: Letter templates, rights information, and more
- **News & Updates**: Stay informed about disability rights and policy

# Features Guide

Complete guide to all features in the Disability Advocacy app.

## üìö Resources Library

### Viewing Resources

The Resources section displays all available disability-related resources. You can:

- **Browse by Category**: Use the category filter to view resources by type
- **Search**: Type in the search bar to find specific resources
- **View Details**: Tap any resource to see full information including:
  - Description
  - Category and tags
  - Website link
  - Related resources

### Managing Favorites

- **Add to Favorites**: Tap the heart icon on any resource
- **View Favorites**: Access your favorites from your Profile
- **Remove from Favorites**: Tap the heart icon again to remove

### Categories

Resources are organized into categories:
- Legal Rights
- Education
- Employment
- Healthcare
- Technology
- Community Support
- Government Services
- Advocacy Organizations

## üìÖ Events

### Viewing Events

- **List View**: See all upcoming and past events
- **Filter by Category**: Use the category picker to filter events
- **Event Details**: Tap an event to view:
  - Full description
  - Date and time
  - Location (virtual or in-person)
  - Accessibility information
  - Registration link

### Calendar Integration

- **Add to Calendar**: Tap "Add to Calendar" to add events to your system calendar
- **Reminders**: Events added to your calendar will use your system reminder settings

## üîç Search

### Unified Search

Search across all content types in one place:

- **Resources**: Find resources by title, description, or tags
- **Events**: Search events by title, description, or location
- **Community Posts**: Search community discussions

### Search Features

- **Real-time Results**: Results update as you type
- **Filter by Type**: Filter results to show only Resources, Events, or Posts
- **Recent Searches**: Your recent searches are saved
- **Clear Search**: Tap the X to clear your search
